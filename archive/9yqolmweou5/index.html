引擎 V8 推出“并发标记”，可节省 60%-70% 的 GC 时间

这项技术名为“并发标记（ concurrent marking）”，在 GC 扫描和标记活动对象时，允许 JavaScript 应用程序继续运行。测试显示，并发标记技术为主线程标记节省了 60％-70％的时间。
